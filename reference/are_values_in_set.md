# Stop statement values are not in set

Stop statement values are not in set

## Usage

``` r
are_values_in_set(
  df,
  cols,
  set,
  main_message = "All columns must be in the following set: "
)
```

## Arguments

- df:

  A data frame

- cols:

  A vector of column names (quoted)

- set:

  A vector of values

- main_message:

  A main message

## Value

A stop statement
