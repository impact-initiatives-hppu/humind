# Stop statement "If not in colnames" with colnames

Stop statement "If not in colnames" with colnames

## Usage

``` r
if_not_in_stop(df, cols, df_name, arg = NULL)
```

## Arguments

- df:

  A data frame

- cols:

  A vector of column names (quoted)

- df_name:

  Provide the tibble name as a character string

- arg:

  Default to NULL.

## Value

A stop statement
